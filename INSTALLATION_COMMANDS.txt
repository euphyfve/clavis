  CLAVIS WORDBOARD - INSTALLATION COMMANDS
═══════════════════════════════════════════════════════════════

📋 COPY AND PASTE THESE COMMANDS IN ORDER

──────────────────────────────────────────────────────────────
STEP 1: Navigate to Project Directory
──────────────────────────────────────────────────────────────
cd c:\xampp\htdocs\clavis-pasha

──────────────────────────────────────────────────────────────
STEP 2: Install PHP Dependencies
──────────────────────────────────────────────────────────────
composer install

──────────────────────────────────────────────────────────────
STEP 3: Install Node Dependencies
──────────────────────────────────────────────────────────────
npm install

──────────────────────────────────────────────────────────────
STEP 4: Generate Application Key
──────────────────────────────────────────────────────────────
php artisan key:generate

──────────────────────────────────────────────────────────────
STEP 5: Create Storage Link
──────────────────────────────────────────────────────────────
php artisan storage:link

──────────────────────────────────────────────────────────────
STEP 6: Create Database
──────────────────────────────────────────────────────────────
Option A: Using MySQL Command Line
mysql -u root -p
CREATE DATABASE clavis_wordboard CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
EXIT;

Option B: Using phpMyAdmin
1. Go to http://localhost/phpmyadmin
2. Click "New" in the left sidebar
3. Database name: clavis_wordboard
4. Collation: utf8mb4_unicode_ci
5. Click "Create"

──────────────────────────────────────────────────────────────
STEP 7: Run Migrations
──────────────────────────────────────────────────────────────
php artisan migrate

──────────────────────────────────────────────────────────────
STEP 8: (Optional) Seed Demo Data
──────────────────────────────────────────────────────────────
php artisan db:seed

This creates 3 demo users:
- alex@clavis.test (password: password)
- nova@clavis.test (password: password)
- zara@clavis.test (password: password)

──────────────────────────────────────────────────────────────
STEP 9: Start Development Servers
──────────────────────────────────────────────────────────────

Open TWO separate terminal windows:

Terminal 1 - Laravel Server:
php artisan serve

Terminal 2 - Vite Dev Server:
npm run dev

OR use single command (requires concurrently):
composer dev

──────────────────────────────────────────────────────────────
STEP 10: Visit Application
──────────────────────────────────────────────────────────────
Open browser and go to:
http://localhost:8000

──────────────────────────────────────────────────────────────
🐛 TROUBLESHOOTING
──────────────────────────────────────────────────────────────

Issue: "Class not found"
Solution: composer dump-autoload

Issue: "Vite manifest not found"
Solution: Ensure npm run dev is running

Issue: "Database connection error"
Solution: Check .env database credentials

Issue: "Storage link already exists"
Solution: This is fine, continue

Issue: "Port 8000 already in use"
Solution: php artisan serve --port=8001
